In this test case we test the ability of a two-\/layer neural network to learn the XOR gate. 

The XOR gate is a binary operation that returns true if the inputs are different, and false otherwise. The network consists of two layers\+: a hidden layer with an OR and an AND neuron, and an output layer with a {\itshape XOR} and a carry neuron.

The XOR in this case is not a traditional XOR gate, but a neuron that computes the XOR operation. It works because it can only take 3 inputs which is linearly separable. The inputs are the output of the OR and AND neurons. The are as follows\+: ~\newline
