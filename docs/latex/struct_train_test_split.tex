A structure to hold the features and targets read from a CSV file. This is for the digit data set. 

This structure contains two members\+:
\begin{DoxyItemize}
\item images\+: A Vector of ints where each 64 elements represent an image of a digit(8x8 image).
\item targets\+: A vector of integers where each element represents the target value corresponding to the features.
